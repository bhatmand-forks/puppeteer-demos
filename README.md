# Puppeteer can automate that!

All the demos that are shown in the talk can be found in this repo.

## How to run scripts?

Install dependencies
```
npm i
```

Execute scripts with
```
node fps.js
```

Run tests with
```
npx jest dom-snapshot.spec.js

```
